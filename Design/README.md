Cyjb.Compilers.Compile
====

提供编译的设计时模板功能，基于 .NET 6。

本项目支持在设计时通过 [T4 模板](https://docs.microsoft.com/zh-cn/visualstudio/modeling/code-generation-and-t4-text-templates?view=vs-2022)，
生成词法分析器和语法分析器。

详细的类库文档，请参见 [Wiki](https://github.com/CYJB/Cyjb.Compilers/wiki)。

欢迎访问我的[博客](http://www.cnblogs.com/cyjb/)获取更多信息。
